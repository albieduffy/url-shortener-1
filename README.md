# URL Shortener App

Created using Python and Flask

## Instructions

Run `pipenv install` and `pipenv shell`.

Then run `flask run`.

Followed [Pretty Printed's](https://www.youtube.com/watch?v=rGQKHpjMn_M&ab_channel=PrettyPrinted) tutorial
